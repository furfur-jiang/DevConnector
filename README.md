# DevConnector

基于react express mongodb技术

## 基本功能

- [x] 开发者总览
- [x] 开发者个人信息：链接获取github仓库
- [x] 简介/经历/教育 增删改查
- [x] 社区留言，回复留言
- [x] 登录/注册/退出
